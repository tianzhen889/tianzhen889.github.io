ParticleX主题设置iconLinks

2023/12/2 前端网页 Hexo ParticleX

对于ParticleX主题的iconLinks，其实作者在评论区详细介绍过，这里只是简单强调一下

其实和配置文件中menu的配置一样：

menu:
    Home:
        name: house
        theme: solid
        link: /
    About:
        name: id-card
        theme: solid
        link: /about
    Archives:
        name: box-archive
        theme: solid
        link: /archives
    Categories:
        name: bookmark
        theme: solid
        link: /categories
    Tags:
        name: tags
        theme: solid
        link: /tags

 iconLinks:
        github:
            name: github
            theme: brands
            link: https://github.com/tianzhen889/
        bilibili:
            name: bilibili
            theme: brands
            link: https://space.bilibili.com/1831642830?spm_id_from=333.1007.0.0
        qq:
            name: qq
            theme: brands
            link: tencent://AddContact/?fromId=45&fromSubId=1&subcmd=all&uin=2647298256&website=www.oicqzone.com
        zhihu:
            name: zhihu
            theme: brands
            link: https://www.zhihu.com/people/ninth-84

如果你想获取一个新图标，只需在Font Awesome官网查找对应的图标即可

然后按照我上面的对齐格式，比如说qq：

<i class="fa-brands fa-qq"></i>

那么theme中就填brands，name中就填qq，链接你可以在网上找一个加好友的链接。

其他的图标类似。
